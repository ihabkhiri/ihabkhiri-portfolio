# Ihab Khiri - Personal Portfolio

Welcome to the repository for my personal website and portfolio. This site showcases my experience as a Microsoft Ecosystem SME (Microsoft 365, Power Platform, Dynamics 365 & Azure AI Copilot) and hosts my personal blog.

Live site: [https://your-live-domain.com](https://your-live-domain.com)

## Tech Stack

This project is built with modern web technologies:
- **Framework:** Next.js 15 & React 19
- **Styling:** Tailwind CSS v4
- **Animations:** Motion
- **Content:** MDX for blog posts

## Running Locally

If you want to run this project on your local machine, follow these steps:

```bash
# 1. Clone the repository
git clone [https://github.com/ihab-khiri/ihabkhiri-portfolio.git](https://github.com/ihab-khiri/ihabkhiri-portfolio.git)

# 2. Navigate to the project directory
cd ihabkhiri-portfolio

# 3. Install the dependencies
npm install

# 4. Start the development server
npm run dev
